# resumo-lab-az900
Resumo para entrega na plataforma

No lab, foi explorado no portal da azure as categorias de recursos disponíveis para uso, além de ter explicado um pouco melhor sobre alguns recursos como o Bastion, foi reforçado a variedade de funções disponíveis e o fato que o conhecimento de computação em nuvem é necessário para praticamente todos os profissionais de TI, seja trabalhando com IA, desenvolvimento, infraestrutura, etc... Também foi falado um pouco sobre recursos em versão prévia e os riscos de seu uso em produção.
